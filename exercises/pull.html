Muscles Worked in Pulling Motions

Pulling motions work what would be considered muscles on the posterior of the body, with secondary muscles on the anterior worked as well.

What muscles are worked in a pull up

  • Lats
  • Posterior Deltoids
  • Traps (middle and lower)
  • Biceps
  • Infraspinatus (External Shoulder Rotator)
  • Forearm muscles
  • Core

When planning out a "pull day", I have found it best to order pushing movements based on orientation, supersetted each with a core exercise. This gives about 4 pull exercises and 2-4 core exercises. You can add 2 accessory exercises for the smaller muscle worked, as well! This is completely optional.

Sample Pull Day

  1. Vertical Pull: Pull Ups
  2. Horizontal Pull: Bent Over Row
  3. Core: Plank
  4. Vertical Pull: Lat Pulldown
  5. Horizontal Pull: Inverted Row
  6. Core: Russian Twist
  7. Accessory (Optional): Biceps Curl
  8. Accessory (Optional): Preacher Curl
